# The Catalog - Library Management System

A full-stack library management system with a hosted Flask backend and a modern HTML/CSS/JavaScript frontend.

## Getting Started

Simply open `frontend/index.html` in your web browser.

## Project Structure

```
The-Catalog/
├── backend/          # Flask API server (hosted)
├── frontend/         # HTML/CSS/JavaScript client
│   └── index.html    # Main entry point
├── docker-compose.yml # Docker configuration (for development)
└── init.sql          # Database initialization script
```

## Technology Stack

- **Backend:** Flask, PostgreSQL, Python (Hosted on **Google Cloud Run**)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)

## Troubleshooting

- **API connection issues:** Ensure you have an active internet connection as the backend is hosted remotely
